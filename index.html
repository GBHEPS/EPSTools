📄 About This Suite

These tools are built for internal use by East Portland Sash field and shop staff. No app install required — just open in any browser and go. Data syncs across machines via Firebase where applicable.

For best results use Chrome or Safari. If something looks off, try a hard refresh (Cmd+Shift+R on Mac).
